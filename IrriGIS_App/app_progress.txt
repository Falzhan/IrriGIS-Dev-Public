RECENT UPDATES (v1.2.6)
===============================================================================
[x] Draft / Offline Report System (NEW)
    - photo capture + auto-save draft at the moment photo is taken
    - draft saved to local pending_reports.json before any submit attempt
    - Tap "Submit" while online → draft deleted from local and report sent to server
    - Tap "Submit" while offline → draft stays, reports appear in Me tab, auto-sync later
    - Tap ✕ / Retake on form → in-flight draft silently deleted
    - App crash / force-close while form is open → draft survives restart and appears in Me tab

    - Draft cards prepended before every server report/ticket in Me view

    - app/(tabs)/camera.tsx: useFocusEffect rehydrates all form fields

PART 6: Network Configuration
================================================================================

For local testing:
  - Update EXPO_PUBLIC_API_URL in .env to your local IP (e.g., http://192.168.1.192:3000/api)
  - Restart Expo with: npx expo start --clear

For deployment:
  - Already configured to use: https://irrigis-backend.onrender.com/api
  - CORS configured to allow render.com domains
